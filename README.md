### Check CryptoCurrency Coin Rate TeleGram Bot

#### Build 

- You Have to build image with this command
 
```s
docker-compose build
```

#### Configure


- Use ENV in `docker-compose.yml`


```s
TELEGRAM_BOT: 'your_token'
```


#### Run


```s
docker-compose up -d 
```

#### How to use

- Use `check` command with args in channel
